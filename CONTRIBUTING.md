# 贡献指南

欢迎为 BumbleCore 做出贡献。

---

## 如何贡献

### 报告 Bug

在 [Issues](https://github.com/wxhcore/bumblecore/issues) 中提交 Bug 报告，包含：
- 复现步骤
- 期望与实际行为
- 环境信息（Python、PyTorch、CUDA 版本等）
- 错误日志

### 功能建议

提交 Feature Request 前，请先搜索是否已有相关讨论。说明使用场景、预期设计和实现思路。

### 提交代码

大的改动请先在 Issue 中讨论。提交前确保代码通过所有检查。

---

## 开发环境

### 安装

```bash
git clone https://github.com/YOUR_USERNAME/bumblecore.git
cd bumblecore
git remote add upstream https://github.com/wxhcore/bumblecore.git
```
